Algorithm Analysis - Task

You'll have to code in Python the "Guess the Number" game.
Here: Guess the Number @inventwithpython.com you can find the explanation and sample code.
Your task is to write the program, such that the computer is the "Player" - i.e. the one who is going to guess.
And to evaluate the algorithm you've used!
